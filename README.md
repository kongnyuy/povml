This folder contains the analysis code and the visualization source code.

## Sub-Projects

This repository is organized into several sub-projects, each handling a specific stage of the thesis workflow:
- **`analysis`**: Scripts and tools for gathering raw datasets from various APIs and web sources. It equally contains Data cleaning, normalization, and feature engineering pipelines to prepare the raw data for model training.

- **`application`**: This folder contains code necessary for conducting interactive evaluation of the study. It uses streamlit.
